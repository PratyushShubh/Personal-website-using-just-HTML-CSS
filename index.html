Arduino Based Digital IC Tester

I developed an Arduino-based IC Quad Tester on Tinkercad that simplifies the process of verifying integrated circuits. By inserting the IC into the designated position, the tester automatically checks its functionality and identifies the IC name. It systematically tests various logic gates like AND, NAND, OR, NOR, XOR, and XNOR, providing an efficient and user-friendly tool for electronics enthusiasts to validate and diagnose ICs accurately. This project enhances the ease and reliability of working with integrated circuits.

Autonomous Vehicle Motor Control System

The motor control system for an autonomous vehicle, created in Tinkercad, leverages an Arduino Uno and L298N motor drivers to achieve precise navigation and movement. Integrating various sensors for obstacle detection and path following, the system ensures smooth and accurate control over the vehicle's motors. This innovative project demonstrates a practical application of autonomous technology, providing a robust framework for future enhancements and real-world deployment in automated transportation.
